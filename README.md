# Simonize

## Overview
Simonize also called Simone Says is a classic memory game where players must repeat a sequence of colors. The game challenges players to remember and replicate increasingly longer sequences, testing their memory and attention skills.

## Technologies Used
- *HTML*: For the structure of the game interface.
- *CSS*: For styling and layout.
- *JavaScript*: For game logic and interactivity.

## Features
- Interactive color buttons that light up in a sequence.
- Increasingly difficult rounds as players successfully complete sequences.
- Score tracking to keep track of player performance.
- Simple and intuitive user interface.
